## 🚀 Как запустить

### 1. Установите Node.js
Скачайте с https://nodejs.org (версия 18 или выше)

### 2. Откройте терминал в папке проекта
```bash
cd my-app
npm install
npm start
```
Приложение откроется на http://localhost:3000

### 3. Сборка для деплоя на хостинг
```bash
npm run build
```
Папка `build/` готова для загрузки на Vercel или любой хостинг.

## 📱 Установка на iPhone как PWA

1. Задеплойте на **Vercel** (бесплатно):
   - Зарегистрируйтесь на https://vercel.com
   - Перетащите папку `my-app` в Vercel Dashboard
   - Получите ссылку вида `https://my-app.vercel.app`

2. Откройте ссылку в **Safari** на iPhone
3. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой ↑)
4. Выберите **«На экран "Домой"»**
5. Нажмите **«Добавить»**

Приложение появится как иконка на рабочем столе и будет работать как нативное!

## 🔄 Как работает синхронизация

```
Слава вносит данные на iPhone
         ↓ мгновенно
   Firebase Cloud (Google)
         ↓ мгновенно
Анечка видит изменения на своём iPhone
```

- 🟢 Зелёная точка в шапке = данные синхронизированы
- 🟡 Жёлтая точка = идёт подключение
- Без интернета данные сохраняются локально и синхронизируются при появлении связи

## 🔒 Правила безопасности Firebase

Откройте Firebase Console → Realtime Database → Правила и вставьте:

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

Это открытые правила для личного семейного использования.
