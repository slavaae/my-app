# 💑 Мы — Семейный бюджет

## Как запустить локально

### Требования
- Node.js 18+ (скачать на https://nodejs.org)

### Установка и запуск
```bash
npm install
npm start
```
Откроется браузер на http://localhost:3000

### Сборка для деплоя
```bash
npm run build
```
Папка `build/` готова для загрузки на хостинг.

## Деплой на Vercel (бесплатно)
1. Зарегистрируйтесь на https://vercel.com
2. Нажмите "Add New Project" → "Import Git Repository"  
   ИЛИ используйте Vercel CLI: `npx vercel`
3. Vercel автоматически определит React и задеплоит
4. Получите ссылку вида https://my-app.vercel.app

## Установка на iPhone как PWA
1. Откройте ссылку приложения в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Выберите «На экран "Домой"»
4. Нажмите «Добавить»
Приложение появится как иконка на рабочем столе!
